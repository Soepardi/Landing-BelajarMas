# Proyek Ujian Pemrograman Berbasis Web

## Deskripsi
Proyek ini dibuat sebagai **tugas UTS** mata kuliah **Pemrograman Berbasis Web**.  
Aplikasi berupa **website statis / landing page** yang dibangun menggunakan **HTML, CSS, dan JavaScript**.

##Teknologi yang Digunakan
- **HTML5** – Struktur dan konten website  
- **CSS3** – Styling dan layout website  
- **JavaScript** – Interaksi dan logika pada website
